Currency API
- This project demonstrates automated an currency API using Python.A request is made by using a API_KEY.After all the requiriments for the request be made
the informations of that API are converted in json,then converted in Exel format using Pandas.This project shows the current Brazil's currency rate in dollar,
the last update, and when will be the next update.
